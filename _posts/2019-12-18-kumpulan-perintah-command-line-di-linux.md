Perintah dasar di linux.

Berikut dibawah ini :

1. Membuat direktori.
```sh
$ mkdir namafile
```
2. Melihat isi direktori.
```sh
$ ls
```
```sh
$ ll
```
3. Merename direktori.
```sh
$ mv namafile namafilebaru
```
4. Memindahkan direkotri.
```sh
$ mv namafile /home/sandy/
```
